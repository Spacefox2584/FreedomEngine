# FreedomEngine (FE)

FreedomEngine is a browser-native, local-first operations OS.
It runs as a single web app and hosts modular tools (“GIZMOs”)
inside a fast, inspectable shell.

## Current status
- Release: FE.01.A1.R2
- State: Active development
- Persistence: Local-first (IndexedDB), no server sync yet

## Repo structure
/core      — FE shell, routing, state, persistence
/gizmos    — Modular tools (C-Deck, Planner, etc)
